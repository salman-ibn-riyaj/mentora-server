# ⚙️ Mentora - Server Side

Welcome to the backend repository of **Mentora**. This is a robust and scalable REST API built with Express.js and MongoDB, powering the Mentora platform to connect mentors and mentees seamlessly.

🔗 **Live API Production URL:** [https://montera-server.vercel.app/](https://montera-server.vercel.app/)

---

## 🛠️ Tech Stack

This server-side application utilizes the following technologies:

- **Runtime Environment:** [Node.js](https://nodejs.org/)
- **Framework:** [Express.js](https://expressjs.com/) (Fast, unopinionated, minimalist web framework)
- **Database:** [MongoDB](https://www.mongodb.com/) (NoSQL Database)
- **ODM:** Mongoose (for MongoDB object modeling)
- **Icons & Frontend Support:** React Icons (on the client side)

---
